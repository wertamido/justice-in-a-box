# PART 1: THE FULL EXPLANATION — "Justice in a Box"

## What This Tool Does

"Justice in a Box" transforms the chaotic, traumatic evidence that abuse victims collect — screenshots of threatening texts, photos of injuries, voice messages, medical records, witness accounts — into a **professional, court-ready legal timeline** that lawyers and judges can use immediately.

## The Problem It Solves

When victims of domestic violence finally decide to seek justice, they face a devastating paradox: they have YEARS of evidence (hundreds of texts, voicemails, photos) but it's completely **disorganized**. They show up at the police station or court with:
- Screenshots scattered across 10 different apps
- Voice messages they can't transcribe
- Photos with no dates or context
- Emotional, scattered testimony that doesn't match legal standards

**Result**: Judges reject messy evidence. Police don't take them seriously. Cases get dismissed.

## How "Justice in a Box" Works (Step by Step)

### Step 1: Upload Everything (No Judgment, No Organization Needed)
The victim simply drops ALL their evidence into the tool:
- Screenshots of abusive text messages (WhatsApp, SMS, Messenger, Instagram...)
- PDFs of medical records, police reports, witness statements
- Audio files of voicemails, recorded arguments, threatening calls
- Photos of injuries, property damage

### Step 2: AI Extracts & Organizes Everything Automatically
- **Tesseract.js (Free OCR)** reads text from screenshots — even in different languages
- **Web Speech API / Whisper (Free)** transcribes audio files to text
- **Mistral AI (Free tier)** analyzes all content, extracts dates, identifies legal keywords

### Step 3: AI Generates Court-Ready Legal Timeline
Mistral transforms chaotic evidence into:
- Chronological timeline with exact dates and times
- Legal terminology ("coercive control," "psychological violence," "physical assault")
- Categorized evidence types (threats, physical violence, financial abuse, isolation)
- Professional PDF formatted for court submission

### Step 4: Export & Use
- Download the PDF
- Share with lawyer
- Submit directly to court or police
- Available in French (for France/Belgium/Switzerland/Canada) + other languages

## Why It's Revolutionary

| Traditional Process | Justice in a Box |
|---|---|
| Victim pays €3,000-€8,000 for lawyer to organize evidence | **100% FREE** |
| Takes 3-6 months to prepare case file | **Done in 10 minutes** |
| Victim must relive trauma by manually writing everything | **AI handles it, victim just uploads** |
| Evidence rejected for being messy/incomplete | **Professional legal format** |
| Victim needs tech skills to organize files | **Drag & drop, one click** |

# PART 3: TECHNICAL ARCHITECTURE (100% FREE)

## How Everything Works For Free

| Feature | Free Technology | Cost |
|---------|---------------|------|
| OCR (text from images) | Tesseract.js (client-side) | $0 |
| Audio transcription | Web Speech API (browser built-in) | $0 |
| AI reasoning & legal analysis | Mistral API (free tier) | $0 (up to 1M tokens/month) |
| PDF generation | jsPDF + html2canvas (client-side) | $0 |
| Storage | localStorage / IndexedDB (device only) | $0 |
| Hosting | GitHub Pages / Netlify / Vercel | $0 |
| HTTPS | Provided by hosting | $0 |

## Architecture Diagram

```
User Device (100% Private)
├── Browser
│   ├── HTML + CSS + JavaScript
│   ├── Tesseract.js → OCR from screenshots
│   ├── Web Speech API → Audio transcription
│   ├── localStorage → Evidence storage (encrypted)
│   └── jsPDF → PDF generation
│
│   └── ONE external call (only when analyzing):
│       └── Mistral API (text only, no files uploaded)
│           └── Returns: Legal timeline, categorized evidence
│
└── NO server. NO database. NO backend. 100% FREE.
```

## Privacy & Security

- **No files uploaded to any server** — OCR and transcription happen on the user's device
- **Only extracted text sent to Mistral** — not original images or audio
- **API key stored locally** on user's device only
- **All evidence stays on device** — can be exported as encrypted file
- **No tracking, no analytics, no cookies** — complete privacy
